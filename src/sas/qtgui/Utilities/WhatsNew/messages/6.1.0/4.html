What's New: Shape2SAS

Shape2SAS allows the user to construct real space models from simple building blocks and generate the resulting SAS curve

Shape2Sas allows the user to construct arbitrary objects by adding any number of shapes from the available menu (currently 10). The sizes, sld, placement in space, and orientation in space of each shape (subunit) can be adjusted. The resulting object can then be visualized, and the expected SAS scattering pattern generated.

The current version is labelled "experimental" as it has not been thoroughly tested and still contains some known issues such as the aspect ratio of the real space visualization of the constructed object. Moreover, the ability to create a plugin model with the parameters of the individual building blocks as possible fitting parameters is currently disabled. We look forward to feedback from the community to improve this tool for the next release.
